专为真实项目推进设计。

不是又一个待办清单。Project Flow 更像一张可视化作战图，让任务、日期、分支和完成状态保持在同一个上下文里。

01

时间线画布

所有项目节点按日期展开，今日列表、Deadline 和状态变化会在同一张画布上呈现，减少来回切换。

02

Continue 分支

每一次 Continue 都代表项目向下一步推进。你可以保留不同分支，并通过平移让多组项目保持整齐。

03

Done 收尾

完成节点可以明确打上 Done 标记，让项目不只是开始，还能被清晰地结束、复盘和归档。

04

账号和试用

注册后自动进入 30 天免费试用。永久会员通过 Creem 支付与 License Key 激活，流程更适合正式发售。

05

本地优先

桌面版优先服务 Windows 本地使用场景；账号登录后可配合后端同步，兼顾私有数据和多设备访问。

06

清爽无负担

界面克制、操作直接，适合每天反复打开。你只需要关心项目下一步，而不是管理工具本身。
